# 📊 Sales Analysis Dashboard — Excel

![Dashboard Preview](dashboard-preview.png)

## 📝 Overview

An interactive **Sales Analysis Dashboard** built entirely in **Microsoft Excel** using **Pivot Tables** and **Pivot Charts**. The dashboard provides a comprehensive view of sales and profit performance across categories, sub-categories, regions, and months — all filterable with dynamic slicers.

---

## ✨ Features

- 📦 **Total Sales by Sub-Category** — Bar chart comparing sales across all product sub-categories, highlighting top performers (Copiers: $55K) and loss-makers (Tables: -$17K).
- 🥧 **Total Profit by Category** — Pie chart breaking down profit share:
  - Technology: **50.79%**
  - Office Supplies: **42.77%**
  - Furniture: **6.44%**
- 🗺️ **Regional Breakdown** — Clustered bar chart comparing performance across **Central, East, South, and West** regions for each sub-category.
- 📈 **Monthly Profit Trends** — Line chart revealing seasonality patterns, with profit peaking in **December ($35K)** and dipping in **August (-$21K)**.

## 🚀 How to Use

1. **Open** the Excel file.
2. Navigate to the **`2dashboard pivot`** sheet.
3. Use the **Category slicer** to filter by: `Furniture`, `Office Supplies`, or `Technology`.
4. Use the **Years (Order Date) slicer** to filter by: `2015`, `2016`, `2017`, or `2018`.
5. All charts update **automatically** based on your selections.

## 💡 Key Insights

- **Technology** is the most profitable category, contributing over half of total profits.
- **Tables** is the only sub-category with **negative profit**, suggesting a pricing or cost issue.
- **December** consistently shows the highest monthly profit — ideal for seasonal promotions.
- **August** records the lowest profit dip — worth investigating for operational improvements.
